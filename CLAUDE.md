# Utrecht bus monitor

## Git

- Als een gevraagde wijziging in deze repo is doorgevoerd, commit en push die
  direct naar de bijbehorende remote-tracking branch, zonder daar apart
  bevestiging voor te vragen. Dit geldt voor gewone commits/pushes op de
  huidige branch; force-push, branch-verwijdering en andere destructieve
  git-acties blijven altijd bevestiging vereisen.
